## 📋 Sobre o projeto

O **BarberKing** é uma página web de cardápio digital voltada para barbearias. O cliente acessa, escolhe os serviços que deseja, e visualiza o total antes de confirmar o agendamento.

Projeto desenvolvido com HTML, CSS e JavaScript puro — sem frameworks, sem dependências externas.

---

## ✨ Funcionalidades

- 📌 Listagem de cortes e serviços extras
- ➕ Adição de serviços ao agendamento com um clique
- 💰 Cálculo automático do total
- ✅ Confirmação do agendamento com resumo
- 📱 Layout responsivo (mobile e desktop)
- 🔗 Links diretos para WhatsApp e Instagram

## 🚀 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/erikvictorino/barberking.git
```

2. Abra o arquivo `index.html` no navegador:
```bash
# Ou simplesmente dê um duplo clique no arquivo index.html
open index.html
```

> Nenhuma instalação necessária. Funciona direto no navegador.
